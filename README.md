## INVA Web – Instituto Nuevo Vallarta

Aplicación web desarrollada en **React** para el Instituto Nuevo Vallarta.  
Presenta la oferta educativa, valores institucionales y canales de contacto en una experiencia moderna, rápida y responsiva.

### Características
- SPA con React Router
- SEO optimizado con React Helmet
- Diseño responsive con TailwindCSS
- Botón flotante de WhatsApp
- Integración con Google Maps

### Tecnologías
React · React Router · TailwindCSS · Lucide Icons
